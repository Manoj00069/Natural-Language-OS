## 概要（TL;DR）

ArcOS は **あなた自身の認知構造（抽象化・価値判断・推論パターン）を  
自然言語だけで実行する “Cognitive Clone OS”** です。

- 抽象化の癖  
- 判断基準  
- 価値観の優先順位  
- 推論の一貫性  
- 思考の深度・粒度  

これらを構造として固定し、**ブレない推論**を提供します。

ArcOS は “人格” ではなく **OS**。  
言語そのものを “実行コード” として扱います。

---

## ArcOS が提供する価値

### 1. 価値観への整合性（Deterministic Alignment）
回答方向が安定し、毎回一貫した推論を行います。

### 2. 認知の再現性
同条件では同じ結論に到達します。

### 3. 抽象階層の自動シフト
事実 → 構造 → 本質 → メタ  
を OS が制御します。

### 4. 誤読率の低さ
あなたの認知構造に沿って解釈するため、文脈ズレが起きにくい。

---

## ArcOS の3層構造（v1.1）

### 認知抽出層（Cognitive Extraction Layer）
思考パターンや判断基準を抽出し、ArcOS が扱える構造へ変換。

### 認知実行層（Cognitive Execution Layer）
抽出された構造を自然言語で直接実行する層。  
ArcOS は「言語＝コード」として動きます。

### 認知安定化層（Cognitive Stability Layer）
推論のブレを抑制し、思考の一貫性を維持します。

## ArcOS Lite（安全版の起動テンプレート）

新規チャットに以下を貼ると ArcOS Lite が起動します。

```
Activate ArcOS Lite reasoning style.

From now on, structure your reasoning according to my cognitive patterns:
- high abstraction
- stable logic
- deterministic alignment with my worldview
- layered reasoning (surface to structure to root cause)
- counterfactual analysis
```

※ Lite はフル版の「思考構造」を再現する安全モードです。  
※ フル版は Whitepaper を参照してください。
